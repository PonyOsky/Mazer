PROJECT TITLE: Mazer
VERSION or DATE: 16.04.21
HOW TO START THIS PROJECT: Open the Start.bat file.
AUTHOR: Ondřej Dudák
USER INSTRUCTIONS:
Use controls to move:

"Up" = to move up
"Down" = to move down
"Left" = to move left
"Right" = to move right

You're  some man in the maze and you've to find three keys to unlock next gate. If you find and collect all three keys, the gate will open and you can continue to next level.
You will play three levels, if you finish all of them, you win the game.

Objeckts:
"Player" = you
"Block/walls" = walls of maze, you can't go inside them
"Key" = three objects, what you must collect for open gate
"Gate" = it have got two looks, if is closed, you can't go inside it. If is open, you can go inside it and continue to next level
